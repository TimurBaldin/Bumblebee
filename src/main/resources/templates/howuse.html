Алгоритм создания тестовых данных

  1. Нажать на пункт меню "Создать тестовые данные"
  2. Ввести название колонки ---> нажать на кнопку "Добавить колонку"
  3. Задать параметры для необходимых проверок ----> нажать на кнопку "Сохранить проверку"
  4. Нажать на кнопку "Сохранить колонку". При необходимости добавить в отчет колонку ,повторить действия с шага 2
  5. Нажать на кнопку "Сохранить модель"
  6. Заполнить параметры отчета (Report Excel/Report CSV) ----> нажать на кнопку "Загрузить"
