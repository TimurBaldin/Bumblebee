Сервис позволяет сгенерировать тестовые данные по заданным шаблонам и выгрузить в формате xlsx и csv

Проверка
Входные параметры
Описание
Boundary test lines
Len : Длина строки QUANTITY : Требуемое превышение длины строки Low : Наличие строчных букв Cap : Наличие заглавных букв NullValue : Наличие null значения
Генерирует строки длиной от 1 до Len + QUANTITY .
Specsymbol test lines
specialLen : Длина строки QUANTITY : Требуемое превышение длины строки escSpecial : Наличие esc- специальных символов special : Наличие прочих специальных символов
Генерирует строки специальных символов длиной от 1 до specialLen + QUANTITY .
Boundary test int
IntEnd: Конец числового диапазоне IntStart : Начало числового диапазоне QUANTITY: Количество случайных чисел в рамках диапазона
Генерирует числа для проверки "граничных значений"
Range test int
MaxIntVal: Максимальное число из диапазоне MinIntVal : Минимальное число из диапазоне
Генерирует случайные числа MaxIntVal - MinIntVal
